// installation_guide

Step-by-step setup

Get Testara running on your computer in under 5 minutes. No coding knowledge required.

0
Prerequisites

Make sure you have these installed on your computer before starting:

📦 Node.js v18+ 🔀 Git

Download Node.js from nodejs.org (LTS version recommended). Git comes pre-installed on Mac/Linux; Windows users can get it from git-scm.com.

# Verify installations: $ node -v # Should show v18+ or v20+ $ npm -v # Should show 9+ or 10+ $ git --version # Should show git version 2.x
1
Clone the Repository

Open a terminal (Command Prompt / PowerShell on Windows, Terminal on Mac/Linux) and run:

$ git clone https://github.com/praveen-2528/mockify.git $ cd testara
2
Install Dependencies

This downloads all required packages (React, Socket.IO, SQLite, etc). Takes about 30-60 seconds:

$ npm install # Wait for it to finish... you'll see "added XXX packages"
3
Start Testara

This single command starts both the backend server and the frontend app together:

$ npm run dev:full # ✅ Server running on http://localhost:3001 # ✅ Frontend running on http://localhost:5173
4
Open in Browser

Your browser should open automatically. If not, go to:

→ http://localhost:5173

Create an account (stored locally), pick an exam type, and start practicing! 🎉

5
Invite Friends (Multiplayer)

For multiplayer on the same WiFi network, share your local IP. Friends open it in their browser — no install needed on their end!

# Your friends open this in their browser: → http://192.168.x.x:5173 # For internet play, use the built-in tunnel feature # in the Multiplayer lobby → "Enable Internet Access"
💡 Tip: Only the host needs to install Testara. Friends just need a browser and the room code!
